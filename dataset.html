This project utilized the Mappillary Traffic Sign Dataset. This is an extensive set of various street signs from across the world, containing data on the type of sign and location. We used roughly 48,000 images in our training sets.

The default Mapillary annotations are formatted in json files which is unusable for Yolo. In order to convert them we wrote a script that converts from json to yolo8 format based on a provided file directory. The files change from looking like the left image to the right image.

There was no open source dataset available to train the country classification model. Instead a web scraper was used to extract JPEG images of signs from over 160 countries from Wikimedia Commons. The images were augmented to artificially increase the size and diversity of the training set. The labels were converted to a binary matrix, with each column representing a country
